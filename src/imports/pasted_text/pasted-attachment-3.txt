General slider behavior
	•	the purple handle can be dragged left and right
	•	dragging updates the position smoothly
	•	the track should visually show different colors:
	•	one color to the left of the handle
	•	another color to the right of the handle
	•	the interaction should feel smooth and premium

Catalog axis behavior

For Catalog:
	•	the axis labels at the bottom remain fixed
	•	they do not move with the handle
	•	they stay anchored in place across the entire slider

Annotate and Model axis behavior

For Annotate and Model:
	•	the bottom value label should move with the slider handle
	•	it should stay visually associated with the handle position
	•	these two are not fixed-axis labels like Catalog

Important

Please preserve this distinction clearly:
	•	Catalog = fixed bottom scale
	•	Annotate / Model = bottom value follows handle
